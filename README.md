# 🛒 ShopEase – React E-Commerce Web Application

**ShopEase** is a modern **React-based eCommerce web application** that allows users to browse products, add them to cart, manage quantities, and send messages through a contact form.

The project demonstrates **React fundamentals, Context API, API integration, routing, and state management**.

---

# 🌐 Live Demo

🚀 **Live Website:**  
https://new-project-gamma-one.vercel.app/

Hosted using Vercel.

---

# 📌 Features

- 🛍 Browse products from API  
- 🛒 Add products to cart  
- ➕ Increase product quantity  
- ➖ Decrease product quantity  
- 🗑 Remove items from cart  
- 💰 Automatic total price calculation  
- 📧 Contact form with EmailJS integration  
- 🖼 Random image gallery using API  
- 📱 Fully responsive modern UI

# ⚙️ Installation & Setup

Follow these steps to run the project locally.

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/shopease.git
```

### 2️⃣ Navigate to project folder

```bash
cd shopease
```

### 3️⃣ Install dependencies

```bash
npm install
```

### 4️⃣ Start the development server

```bash
npm start
```

### 5️⃣ Open in browser

```
http://localhost:3000
```
